# K-PanRef
## Introuction
The first graph-based Korean Pangenome Reference constructed by 13 haplotype-phased Korean genome assemblies and KOREF1-G-TTAGGA, the first complete Korean reference genome.  

&nbsp;

## Files
Minigraph-Cactus output files of the K-PanRef are available at zenodo.  

https://doi.org/10.5281/zenodo.20810335

&nbsp;

## Data
The raw sequence data reported in this project have been deposited in the Korea Sequence Read Archive (KRA) in Korea Bioinformation Center, Korea Research Institute of Bioscience and Biotechnology (KAP242397) that are publicly accessible at 

https://kbds.re.kr/KRA.

&nbsp;

The nucleotide sequence data reported in this project have been deposited in the Korea Nucleotide Sequence Archive (KNA) in Korea Bioinformation Center, Korea Research Institute of Bioscience and Biotechnology (KAP242400) that are publicly accessible at 

https://kbds.re.kr/KNA.
